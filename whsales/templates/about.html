Adarnauth Wormhole Sales

Who made this?

This site was created by Adarnof, most famous for his work continuing Raynoldo Rivera's Alliance Auth project.

Born out of frustration with an older auth base, work began on a series of modular Django apps, the first of which powers this site.

How does it work?

Using SSO tokens, the CREST endpoint for character location is accessed. Before a listing can be created, CREST is used to determine if the character is in a legal system to list for sale.

Character location can only be determined while the character is in-game.

Integration with SSO and CREST is performed using one app of the Adarnauth suite, Adarnauth-EVE-SSO, which will serve as a common backbone to all future work.

The goal of this site is to act as a testing ground for future technologies - an area to prototype work with an active user base. By using this site, you're helping me develop. Thanks!

How do I know I can trust you?

The same way you go about trusting anyone else in EVE - you don't!

If you're concerned about security, you can revoke CREST access to your character's location at any time through the EVE Community site - just look for the entries labelled "Adarnauth Wormhole Sales" and press Delete.

Full source code for this site can be found on my Github repository.
